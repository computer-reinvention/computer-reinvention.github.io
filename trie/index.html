Yes. The section is removed.

And the calls trie does make return structured answers, not pages of bytes to skim.

Why the trie path is shorter.

①

It indexes symbols, not bytes.

grep sees lines. trie_grep sees functions, classes, methods — with their qnames, signatures, one-liners, and inbound/outbound edge counts ranked by centrality.

②

One-liners come with the search.

Every hit ships with a generated synopsis. You don't have to open the file to learn what the function does — the question "is this the right symbol?" is answered in the result row.

③

Callers list = the answer's table of contents.

trie_read returns prose plus every immediate caller and callee with their own one-liners. Looking for a regression test? It's in the callers list, named after the behaviour you're investigating.

tests/test_e2e_sync test_resync_removes_section_when_symbol_deleted
④

Hubs declare themselves.

inbound_count=35 on the top hit isn't decoration — it's a structural signal that this is the place where per-file sync logic lives. The agent skips three speculative reads because the graph already said so.

Every turn, the hook picks up exactly the files just edited and stamps the graph. The MCP queries the next agent runs reflect reality — not yesterday's snapshot, not a stale index.

the thing nobody says about coding agents

Code is the executable form of intent.
Not the explanatory form.

Every time an agent reads cascade.py to answer "what does this do and why," it's reconstructing intent from for loops and variable names — under context pressure, with the wrong abstraction. That's where hallucinations come from. Not a lack of intelligence. The wrong artifact.

Code is prose constrained by rules necessary to make it comprehensible to machines made of molten sand.

The codebase should describe itself in prose, kept in sync by construction, as the primary artifact both humans and agents work against.

Code becomes the executable form of the prose, not its source of truth.

The bigger picture becomes the starting point.

The first move stops being "let me find the function" and becomes "let me understand the shape of this." Blast-radius. Why-is-this-the-way-it-is. Architecture archaeology. The value is in the synthesis, not the lookup. Synthesis becomes the default move.

Code, again.

The artifact people reach for inverts. The artifact people argue about inverts. PR review inverts: you review intent. The diff is just confirmation that the change matches its description. Code stops being the thing under discussion and becomes the thing under generation.

④ ring four · the destination

Vibes, again.

The executable form stops being the primary surface for human work. We don't read it. We don't write it directly. We shape the system at a level above it, and the code conforms. The IDE goes away. Changes are made by gesture. The barrier to contributing to a large unfamiliar system collapses from three months of reading code to an afternoon of exploring the prose and graph.
